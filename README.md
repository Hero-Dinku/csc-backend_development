# CSE Motors - Car Dealership Website

A full-stack web application for CSE Motors car dealership built with Node.js, Express, and EJS.

## Features

- **Responsive Design**: Mobile-first approach with professional styling
- **Vehicle Classifications**: Browse vehicles by type (Custom, Sedan, SUV, Truck)
- **Vehicle Details**: Comprehensive vehicle information with professional images
- **MVC Architecture**: Proper separation of concerns with Model-View-Controller pattern
- **Error Handling**: Custom 404 and 500 error pages
- **Professional UI**: Modern design with real car images from Unsplash

## Technologies Used

- **Backend**: Node.js, Express.js
- **Frontend**: EJS templates, CSS3 with responsive design
- **Styling**: Custom CSS with mobile-first approach
- **Images**: High-quality car photos from Unsplash
